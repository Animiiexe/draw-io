# Draw.io - Collaborative Drawing Platform

A modern, intuitive collaborative drawing platform that enables real-time creativity and seamless teamwork from anywhere in the world.

## üé® Features

### Core Drawing Tools
- **Professional-grade brush system** with adjustable sizes and opacity
- **Custom color picker** with 256 colors and custom palette creation
- **Multiple brush types**: Brush, Pencil, Marker, and Smart Eraser
- **Layer support** with advanced canvas management
- **Undo/Redo functionality** for mistake-free creativity

### Collaboration & Sharing
- **Real-time collaboration** with unlimited team members
- **Live user presence** indicators showing who's online
- **Instant synchronization** across all connected devices

### Cross-Platform Experience
- **Responsive design** optimized for desktop, tablet, and mobile
- **Touch optimization** for seamless mobile drawing experience
- **No installation required** - works directly in your browser
- **Progressive Web App** capabilities for app-like experience

## üöÄ Getting Started

### Prerequisites
- Node.js 16.0 or higher
- npm or yarn package manager
- Modern web browser with HTML5 Canvas support

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/yourusername/draw-io.git
   cd draw-io
   ```

2. **Install dependencies**
   ```bash
   npm install
   # or
   yarn install
   ```

3. **Start the development server**
   ```bash
   npm run dev
   # or
   yarn dev
   ```

4. **Open your browser**
   Navigate to `http://localhost:3000` to start drawing!

### Quick Start
No account required! Simply visit the application and start drawing immediately. Share your canvas URL with others to collaborate in real-time.
